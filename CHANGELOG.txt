PWA Cost Record 更新紀錄

版本 V1_2
- 調整交付版號命名為 VX_Y 格式。
- 新增 version.json，作為 GitHub / 遠端版次比對來源。
- 強化 PWA 自動更新流程：啟動時、回到前景時、重新連線時、固定間隔都會檢查遠端版號。
- 當遠端版號較新時，會自動要求 service worker 更新並重新載入，不需使用者手動清快取。
- service-worker.js 改為對 version.json、version.js、service-worker.js 採 network-first，避免版號被舊快取卡住。
- 更新首頁顯示版本為 V1_2。

版本 V1_1
- 移除財政部 API Key 區塊。
- 新增 Google Drive 備份資料夾 #PWA-Cost-Record 與最近五筆雲端備份選擇。
- 新增分類獨立編輯面板與 icon picker。
- 新增多幣值記錄與日期匯率換算。
- 優化版面配置，提升主畫面可用空間。
