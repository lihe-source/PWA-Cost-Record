PWA Cost Record V1.1 更新摘要

1. 移除設定頁內原有的財政部 API Key 設定區塊，發票匯入保留 CSV 方式。
2. Google Drive 備份/還原改為同步面板模式，會列出最近 5 筆備份，並存放於 #PWA-Cost-Record 資料夾。
3. 設定頁新增「分類」入口，進入大/小分類獨立編輯面板，可新增/改名/刪除/更換 icon。
4. 內建 icon picker，新增分類時會自動帶入預設 icon。
5. 新增多幣別記錄功能：台幣、日幣、人民幣、歐元、美金；新增/編輯記錄時可切換幣值，系統會依日期匯率換算。
6. 統計金額改以 TWD 統一計算，單筆明細仍保留原始幣值顯示。
7. 主畫面與整體版面壓縮上下留白與底部占位，提高 PWA 使用面積。
8. 版本更新為 V1.1。

備註：
- 多幣別換算不需要 AI，也不需要 Gemini API Key。
- Google Drive 同步仍需要使用者自行填入 Web OAuth Client ID。
